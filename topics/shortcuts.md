# VSCode

## 通用快捷键

|快捷键|作用|
|:---|:---|
|Ctrl+Shift+P,F1| 展示全局命令面板|
|Ctrl+P|快速打开最近打开的文件|
|Ctrl+Shift+N|打开新的编辑器窗口|
|Ctrl+Shift+W|关闭编辑器|

# Vim

## 以词为单位的移动

|快捷键|作用|
|:---|:---|
|w|下一个 word|
|W|下一个 word（跳过标点）|
|b|前一个 word b(ackward)|
|B|前一个 word（跳过标点）|
|e|跳到当前 word 的尾端 e(nd)|

## 行移动

|快捷键|作用|
|:---|:---|
|0|当前行开头|
|^|当前行第一个非空字符|
|$|跳到行尾|

## 段落移动

|快捷键|作用|
|:---|:---|
|}|下一段（以空白行分隔）|
|{|上一段（以空白行分隔）|
|%|跳到当前对应的括号上（适用于各种配对符号）|

## 跳跃移动

|快捷键|作用|
|:---|:---|
|/x|搜索 x，然后可以用 n 下一个， N 上一个移动|
|\#|向前搜索光标当前所在的字|
|\*|向后搜索光标当前所在的字|
|fx|在当前行移动到光标之后第一个字符 x 的位置 f(ind)x|
|gd|跳到光标所在位置词(word)的定义位置 g(o)d(efine)|
|gg|到文档顶部|
|G|到文档底部|
|:x|跳到第 x 行(x 是行号)|
|ctrl+d|向下翻页 d(down)|
|ctrl+u|向上翻页 u(p)|

## 删除

|快捷键|作用|
|:---|:---|
|x|删除当前字符|
|dw|删除当前光标下的词 d(elete)w(ord)|
|d$|删除光标之后的该行剩余部分|
|dd|删除当前行|

## 复制粘贴

|快捷键|作用|
|:---|:---|
|yy|复制当前行 y(ank)|
|yw|复制当前光标下的词 y(ank)w(ord)|
|p|粘贴 p(aste)|
|P|粘贴在当前位置之前|

## 高效编辑

|快捷键|作用|
|:---|:---|
|di"|删除""之间的内容，'同理|
|yi(|光标在()之间，则复制()之间的内容|
|vi[|光标在[]之间，则选中[]之间的内容|
|dtx|删除字符直到遇见光标之后的第一个 x 字符|
|ytx|复制字符直到遇见光标之后的第一个 x 字符|

## 标记和宏(macro)

|快捷键|作用|
|:---|:---|
|ma|将当前位置标记为 a，26 个字母均可做标记，mb、mc 等等|
|'a|跳转到 a 标记的位置|
|qa|将之后的所有键盘操作录制下来，直到再次在命令模式按下 q，并存储在 a 中|
|@a|执行刚刚记录在 a 里面的键盘操作|
|@@|执行上一次的 macro 操作|

## 以下命令可以对标点内的内容进行操作

|快捷键|作用|
|:---|:---|
|ci’、ci”、ci(、ci[、ci{、ci<|分别更改这些配对标点符号中的文本内容|
|di’、di”、di( 或 dib、di[、di{ 或 diB、di<|分别删除这些配对标点符号中的文本内容|
|yi’、yi”、yi(、yi[、yi{、yi<|分别复制这些配对标点符号中的文本内容|
|vi’、vi”、vi(、vi[、vi{、vi<|分别选中这些配对标点符号中的文本内容|
> 另外如果把上面的 i 改成 a 可以连配对标点一起操作。
